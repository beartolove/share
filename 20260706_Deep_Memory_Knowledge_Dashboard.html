Deep Memory 記憶儀表板

Deep Memory 新增「產生記憶儀表板」：把知識庫轉成關聯圖表與關鍵字排名。
引用來源：Facebook 貼文　｜　Deep Memory 說明網頁

重點摘要

  • Deep Memory 新增「記憶儀表板」功能，可把知識庫內容視覺化。
  • 使用者之後只要說「產生記憶儀表板」，就能生成關聯圖表與關鍵字排名。
  • 這個功能延伸自作者先前把法律判決做成關聯圖表的經驗，讓記憶管理更有儀式感。
  • 已安裝 Deep Memory 的使用者可更新試用；尚未安裝者可讓 AI Agent 協助安裝。

原文

https://www.facebook.com/share/1FAorXvaAp/

前幾天把法律相關判決做成關聯圖表, 覺得還蠻有儀式感的
於是把之前開源的 Deep Memory 也加上儀表板
之後只要說"產生記憶儀表板"
就能幫你產生知識庫的關聯圖表還有關鍵字排名
有安裝的可以更新來玩玩, 沒安裝的趕快讓你的AI Agent安裝
