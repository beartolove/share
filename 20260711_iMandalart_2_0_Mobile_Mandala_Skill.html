Agent 的世界，已經不是「日新月異」可以形容了。

我覺得應該叫做：

時新日異。

昨天好朋友 Shary Lai 來台中講課，我當然要盡地主之誼。除了吃飯，最重要的就是——聊 AI。

她提到最近論文一直寫不出來，我就建議她試試看用 Codex + Skill 的方式處理。

流程大概是這樣：

1. 先把她 140 份論文 Markdown 初稿丟進去

2. 用我昨天剛發布的 Skill：FIRE 法則 2.0 做語意分析，生成 JSON 檔案

3. 再請 Codex 生成 EPUB，直接在 Apple Books 打開

結果，差不多 10 分鐘，就生成了一本有目錄、有索引權重的電子書。

Shary 看了之後，眼睛都笑到合不攏。

為什麼我說這個時代是「時新日異」？

因為今天我又發布了一個新的 Skill：

iMandalart 2.0

這個 Skill 是我書裡非常重要的一部分。它可以把超級複雜的概念，濃縮成手機上很好閱讀的「純文字九宮格」。

每一格大約 135 字，用最小的畫面，承載最大的思考。

這非常適合現在這個 Agent 時代。

更有趣的是，寫這個 Skill 本身其實很快。

我先請 Codex 規劃 iMandalart 1.0 的改版重點，生成一段純文字，再放到 Obsidian 的九宮格裡編輯。接著把 Markdown 丟回 Codex，讓它完成 Skill 改版。

從 iMandalart 1.0 到 2.0，大概兩分鐘完成。

真正花時間的，反而是把它整理好、上傳到 GitHub、讓大家看得懂、願意使用。

可見，在 AI 時代裡，真正燒 Tokens 的地方，往往不是功能，而是：

讓人看得懂。

讓人想使用。

讓人覺得美。

不論如何，新的 Skill 已經做好了。

歡迎大家到 GitHub 再安裝一次，也順手幫我點一個 Star。

讓我們一起享受在手機上閱讀美麗九宮格的威力。
