重點摘要

  • 這則貼文分享一個可在 ChatGPT Work／Codex 使用的 LINE 靜態貼圖產出 skill，目標是把貼圖生成、規格檢查與上傳包整理流程自動化。
  • 操作流程很直覺：從 GitHub 下載 ZIP，解壓後上傳 skill ZIP 到 ChatGPT Work 的「技能」，再用 @ 呼叫技能進行貼圖製作。
  • 該 skill 會逐步確認貼圖張數、尺寸、角色、風格、文字與權利，並輸出符合 LINE Creators Market 規格的上傳就緒包。
  • 完成後可下載 LINE 上傳 ZIP，內容包含貼圖圖檔、主圖、聊天室縮圖與相關 manifest／QA 報告，方便送審前檢查。
  • 這個案例展示 ChatGPT Work skill 的價值：把一次性的 AI 繪圖教學，轉成可重複使用、可引導、可驗證的工作流程。

操作流程

  1. 從 GitHub 專案下載 ZIP。
  2. 解壓縮後取出內含的 skill ZIP。
  3. 到 ChatGPT Work：個人化 → 外掛程式 → 技能 → +。
  4. 上傳 skill ZIP，並在 Work 模式用 @ 呼叫。
  5. 依照問題引導生成貼圖，最後下載可上傳 Creator Market 的 ZIP。
