重點摘要

  • Cowart 可視為裝進 Codex 裡的「無限畫布」，讓使用者把參考圖、草稿、排版與生成區域放在同一個工作空間。
  • 它切中的痛點是 AI 改圖時常常理解偏掉：只想微調位置或文字，模型卻可能整張重畫、背景也被改掉。
  • Cowart 的做法是讓使用者直接圈選、畫箭頭、寫標註，讓 Codex 依照畫布上的視覺指令產出新版本。
  • 這代表 AI 設計工作流正從純文字 prompt，走向「看畫布、讀標註、保留原圖、並排比較」的真實設計協作。
  • 目前仍屬早期工具，安裝與 API 設定有門檻，也不是 OpenAI 官方插件；但對社群圖、商品圖、簡報素材製作者值得觀察。

今天GitHub又瘋了

這次看到一個很適合常常需要改圖的人收藏的 Codex 插件：Cowart。你可以把它想成「裝進 Codex 裡的無限畫布」，不只可以放參考圖、畫草稿、排版，還能直接叫 AI 在指定位置生成圖片。

很多人用 AI 改圖最痛苦的，不是 AI 不會畫，而是溝通很容易歪掉。你說人物往左一點，它可能整張重畫；你只想改文字，它卻順便換了背景。最後不是在改圖，而是在跟 AI 玩猜謎。

Cowart 解決的方式很直覺。你可以先畫好圖片框，讓 AI 按照框的比例生圖；也可以直接在圖片上圈選、畫箭頭、寫修改內容，再把標註截圖交給 Codex。它會理解你的改稿，產生乾淨的新版本放在原圖旁邊，原圖也會保留。

這個方向真正有意思的地方，是 AI 設計不再只是一直寫提示詞，而是開始像真的設計工作一樣，可以看畫布、讀標註、反覆修改。目前 Cowart 還偏早期，安裝和 API 設定有門檻，也不是 OpenAI 官方插件，但對經常做社群圖、商品圖、簡報素材的人來說，很值得觀察。
