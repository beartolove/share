重點摘要

  • 數位敘事力期刊主編推出 ChatGPT Work「AI 簡報圖 skill」，可用於製作 16:9 簡報圖與 PDF 簡報。
  • 這個 skill 的設計不是直接生成成品，而是先引導使用者填寫頁數、風格與簡報內容，再對齊大綱與排版。
  • 安裝方式是從 GitHub 下載 ZIP，解壓縮後取出指定 skill ZIP，再到 ChatGPT Work 的技能功能上傳。
  • 使用時在 Work 模式透過 @ 呼叫技能，AI 會先產出簡報大綱與範本，經使用者檢視確認後才進入正式生成。
  • 完成後會將簡報打包輸出為可下載的 PDF，適合教育科技、簡報製作與 AI 輔助內容生成情境。

操作流程

下載安裝

到 GitHub 下載 ZIP，解壓縮後取出 skill ZIP，上傳到 ChatGPT 技能功能。

Work 模式

在 ChatGPT Work 模式中透過 @ 呼叫 Visual Presentation Production。

先對齊需求

AI 會引導填寫頁數、風格、簡報內容，並先確認大綱與排版。

檢視後生成

先產出簡報大綱與範本，使用者核准後才開始正式製作。

PDF 輸出

最後打包輸出為 16:9 PDF 簡報，可直接下載使用。
