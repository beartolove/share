重點摘要

  • 小關用 Claude Code 把單點工具升級成一整套可運作的「一人數位公司」：35+ 套系統、11 位 AI Agent 員工、10+ 條排程與 18+ 個 Claude Skills。
  • 系統涵蓋經營指揮、行銷銷售、生產供應鏈、人資財務、對外服務、AI 創作與協作管理，已不是單一 AI 玩具，而是公司級營運網。
  • 最具代表性的案例是 FB 廣告投放全管線：AI 撈數據、診斷、產文案、API 建廣告，人只負責最後按「啟用」。
  • 自動排程讓公司狀況主動推播到手機：員工進度、客戶告警、商品月報、品牌聲量與事件觸發通知都自動跑。
  • 整套方法的核心不是「用 AI 取代人」，而是「寫系統、用 Agent、做自動化」，讓人退到判斷與把關的位置。

五大板塊

板塊一：35+ 套自建系統

從經營戰情中心、銷售分析、廣告儀表板，到生產排程、成本、倉儲、人資財務、LINE 客服與 AI 創作，將公司各環節逐一系統化。

板塊二：AI 直接操作業務

招牌案例是 FB 廣告投放全管線：Claude Code 串接雲端 graph 代理與 FB Marketing API，完成撈數據、診斷、產文案、建廣告，人只最後啟用。

板塊三：全自動排程

雲端排程與本機 Claude 排程並行，早上與整點自動推播公司狀況，包含員工進度、B 端客戶告警、商品月報、品牌聲量與 token 續命。

板塊四：AI Agent 員工體系

8 位 AI 辦公室動物員工＋3 位第二大腦角色，每位背後都是 Claude 子代理，能分工開發、研究、文案、企劃、數據與知識管理。

板塊五：踩雷變制度

把事故轉成 SOP：規則主檔、三道備份、防 key 外洩掃描、三層防護部署、Firebase 分流、收工／開工交接與 DEPLOY.md。
