自動剪片開源工具 video-autopilot-kit：CapCut 與 ffmpeg 兩種路徑

#科技及AI新知#自動化#開源工具#短影音
引用來源：Facebook 貼文　｜　GitHub：Hao0321/video-autopilot-kit

重點摘要

  • 這是一套已開源的自動剪片 skill／工具包，目標是把素材交給系統後，由系統自動分析與剪輯，減少手動進剪輯軟體逐格調整的時間。
  • 專案提供兩條剪輯路徑：一條直接操作 CapCut／CapCut draft JSON，另一條走純程式化 ffmpeg pipeline，可依使用習慣與平台需求選擇。
  • 若素材不足，作者提到可用新訓練好的動態字卡能力補足畫面，讓短影音或教學內容不會因素材少而卡住。
  • GitHub repo「video-autopilot-kit」採 MIT 授權，描述為 YouTube／短影音自動化框架，包含 CapCut JSON、ffmpeg tooling 與 onboarding questionnaire。
  • 作者表示安裝方式可直接把 GitHub 網址交給 Claude 或 Codex，請 AI 協助一步步設定。
