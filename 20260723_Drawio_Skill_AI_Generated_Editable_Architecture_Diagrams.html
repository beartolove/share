drawio-skill：讓 AI 一句話畫好可編輯架構圖

把自然語言、程式碼、Docker Compose、Terraform、SQL schema 等輸入交給 AI，產生可編輯的 draw.io 圖表，並匯出 PNG／SVG／PDF／JPG。

#科技及AI新知#開源工具#drawio#AI架構圖#AgentSkills

重點摘要

  • 一句話產圖：用大白話描述系統架構，AI 會產生對應的 .drawio XML，再透過本地 draw.io CLI 匯出圖片或向量圖。
  • 不只畫圖，也能讀專案：可解析 Python／TypeScript 程式碼、Docker Compose、Terraform、SQL schema，把依賴關係轉成可視化架構圖。
  • 重點是可編輯：輸出不是靜態截圖，而是 draw.io 原生格式，方便後續人工微調、補標註與納入文件。
  • 內建素材庫：GitHub repo 說明支援 10,000+ draw.io shapes／官方圖示，涵蓋雲端、Kubernetes、UML、BPMN、ER 等常見圖形。
  • 安裝門檻：本機需先安裝 draw.io 桌面版，再將技能包加到 Claude Code、Cursor 或其他 Agent Skills 相容助理。

適合拿來做什麼？

架構說明Web 服務、API Gateway、資料庫、快取、佇列與外部 API 的系統圖。
專案盤點讀程式碼與配置，自動整理模組、服務與依賴關係。
DevOps 文件Docker Compose、Terraform、Kubernetes、CI/CD pipeline 視覺化。
教學簡報將抽象架構轉成圖，協助團隊討論與新人理解。

功能對照

輸入可產出價值
自然語言提示詞.drawio XML、PNG、SVG、PDF、JPG快速把架構想法變成可討論圖稿。
Python／TS／Go／Rust 程式碼模組圖、類別階層、依賴圖降低人工讀 code 後再畫圖的成本。
Docker／Terraform／Kubernetes基礎設施與服務關係圖讓 IaC 配置變成文件與溝通素材。
SQL schemaER diagram讓資料表、PK／FK 與關聯更容易檢視。

快速安裝

貼文提到 macOS 可先安裝 draw.io 桌面版：

brew install --cask drawio

再將技能加入 AI 助理：

npx skills add Agents365-ai/365-skills -g

小提醒

這類工具最適合作為「第一版架構圖草稿產生器」：讓 AI 先幫你排版與抽依賴，再由人確認命名、邊界、資安敏感資訊與最終文件品質。若要放進正式文件，建議仍要人工 review 圖中的服務關係與資料流。
